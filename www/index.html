About Prudence

Prudence is a platform for creating scalable web applications and network services using dynamic scripting languages and proven REST principles. Your application can support thin clients (HTML) and rich clients (AJAX), with anything in between. You can also create pure REST services with no user interaction.

See the main site for more information and full documentation.

Releases are available for download here.

Features

In addition to making it very easy to create REST resources (see demos), Prudence allows embedding scriptlets within HTML pages, similarly to PHP, JSP and ASP (see demos), with support for high-performance streaming and caching of results.

You can see your changes happen on-the-fly without any compilation: Prudence picks up your edits and makes sure to update its cached code base. Development is quick, powerful and fun!

Supported Scripting Languages

  • Ruby
  • Groovy
