About Prudence

Prudence is an open source framework and container for scalable web frontends and network services. It comes in several flavors: Python, Ruby, Groovy, Clojure, JavaScript and PHP. Your applications can support rich clients (AJAX), thin clients (pure HTML), and happy mixes between the two. Prudence addresses real-world, practical concerns, from virtual hosting and URI rewriting to flexible server and client-side caching.

Prudence stands out from other web frameworks and containers in that it is designed from the ground up to scale. It adheres to proven REST principles, which work with the web's inherently scalable architecture, rather than trying to awkwardly force it into other paradigms. Code runs on a JVM: dependable, fast and portable.

Each flavor comes with a data-driven demo application, showing how to integrate Prudence with data backends. The detailed tutorial is a great place to start!

All you need is a Java 5 Standard Edition environment or above. Note that no Java coding is necessary. Also, Prudence does not require or use Java Enterprise Edition extensions.

If you do use Java, you will also find Prudence most useful! Prudence works well as a container for your Restlet applications: a terrific, lean REST-centric substitute for servlet/JEE containers and GWT. Prudence is also available as an embeddable, standalone library.

Features

Prudence makes it very easy to create REST resources. In seconds, you can write code to process and answer an HTTP "PUT" or "GET" with a representation of your resource, and attach this code to a URL. Format negotiation with the client is handled automatically.

Additionally, Prudence offers classic HTML generation by letting you embed code "scriptlets" into text documents, similarly to how PHP, JSP and ASP work. Scriptlets can be written in any of the supported programming languages, or you can choose better-performing, more manageable templating languages: Succinct and Velocity are both included. Prudence goes a step beyond similar platforms by offering fine-grained caching of the generated results.

Development is quick, powerful and fun. Prudence picks up and compiles your changes on-the-fly, making sure to update its cached code base. It offers runtime debugging information and robust logging.
