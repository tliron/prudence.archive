About Prudence

Prudence is an open source framework for building scalable web frontends and network services. It comes in several flavors: Python, Ruby, Groovy, Clojure, JavaScript and PHP. Your applications can support rich clients (AJAX), thin clients (pure HTML), and happy mixes between the two. Not just a framework, Prudence also works as a powerful and flexible container for your web site, addressing real-world, practical concerns, from virtual hosting and URI rewriting to caching.

Prudence stands out from other web frameworks in that it is designed from the ground up to scale. It adheres to proven REST principles, which work with the web's inherently scalable architecture, rather than trying to force it into other, limiting paradigms. HTTP is handled via non-blocking I/O, which keeps your application responsive even under high loads. Code runs on a JVM: dependable, fast and portable.

Each flavor comes with a relevant example data-driven application, showing how to integrate Prudence with a data backend. The detailed tutorial is a great place to start!

All you need is a Java 5 Standard Edition environment or above. Note that no Java coding is necessary. Also, Prudence does not require or use Java Enterprise Edition extensions.

For Java developers, Prudence is also available as an embeddable, standalone library. It also works well as a container for your Restlet applications.

Features

Prudence makes it very easy to create REST resources. In seconds, you can write code to process and answer an HTTP "PUT" or "GET" with a representation of your resource, and attach this code to a URL. Format negotiation with the client is handled automatically.

Additionally, Prudence offers classic HTML generation by letting you embed code "scriptlets" into text documents, similarly to how PHP, JSP and ASP work. Scriptlets can be written in any of the supported programming languages, or you can choose better-performing, more manageable templating languages: Succinct and Velocity are both included. Prudence goes a step beyond similar platforms by offering high-performance streaming and fine-grained caching of the generated results.

Development is quick, powerful and fun. Prudence picks up and compiles your changes on-the-fly, making sure to update its cached code base.
