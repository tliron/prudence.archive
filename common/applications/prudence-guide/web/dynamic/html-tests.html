HTML Generation Demos

These demos all do pretty much the same thing, except that each uses a different language. They're here as as examples of how to write scriptlets. Note that at the top of each page you will see a timestamp -- this is used to test caching mode, during which the timestamp should stay constant for 5 seconds, no matter how many times you refresh the page. See the FAQ for an explanation of the difference between caching and streaming mode.
