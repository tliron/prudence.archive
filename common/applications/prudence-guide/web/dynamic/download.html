Five Mouthwatering Flavors

Yum! Pick your language of choice, download, and you're ready to rumble! All flavors come with everything you need to get started, including documentation and a sample, data-driven web application named Stickstick written in the flavor's language. All flavors support both Succinct and Velocity templating languages.

Feeling lost? Start with the tutorial.

  • Prudence "Succulent Python" Edition (20mb)
  • Prudence "Delectable Ruby" Edition (15mb)
  • Prudence "Scrumptious Clojure" Edition (10mb)
  • Prudence "Savory JavaScript" Edition (5.5mb)
  • Prudence "Ambrosial PHP" Edition (5mb) *
  • Prudence "Luscious Groovy" Edition (9.5mb)

* Note: Due to licensing restrictions, Prudence "Ambrosial PHP" Edition does not come with Quercus. You will need to download (9.5mb) and install it yourself. Simply unpack the .war file (it's a standard zip file) and put resin.jar and javaee-16.jar in Prudence's /libraries/ directory.

Everything But the Kitchen Sink

Any Prudence application, written in whatever language, can run in any of the above flavored containers as long as you add the appropriate language runtime libraries to your classpath. We've created the separate flavors in order to make your download smaller and your experience more focused.

The "Kitchen Sink" edition is available in case you want full support for all languages from the get go.

It's also recommended if you want to try out all the various flavors before deciding on one for your product.

Note that this "Kitchen Sink" uses "Savory JavaScript" as its base flavor. You can build your own "Kitchen Sink" with a different base flavor using the development snapshot (see below).

  • Prudence "Kitchen Sink" Edition (50mb)

Minimal Compact

The "standalone" edition contains only the Prudence runtime library itself and none of the dependencies. Useful if you just want to upgrade the Prudence runtime. It's a tiny little download.

  • Prudence "Just Prudence" Edition (300kb)

Development Snapshot

Latest, greatest, but probably not well-tested. Available via Subversion. Easy, quick build with Ant.
