Flavors

Pick your flavor according to the language of your choice, download, and you're ready to rumble. All flavors come with everything you need to get started, including documentation and a sample application.

  • Prudence for Python
  • Prudence for Ruby
  • Prudence for Clojure
  • Prudence for JavaScript
  • Prudence for Groovy

Kitchen Sink

The "kitchen sink" distribution includes all languages. It's a big download! By default, it uses the JavaScript flavor as its base. You can rebuild the "kitchen sink" distribution with another language as its base using the development snapshot (see below).

  • Prudence "Kitchen Sink"

Standalone

The "standalone" distribution contains only only Prudence source code and none of the dependencies. It's tiny.

  • Prudence Standalone

Development Snapshot

Available via Subversion.
